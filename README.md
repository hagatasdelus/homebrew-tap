# hagatasdelus/tap

Third party homebrew tap to install the tools Hagata is involved in developing.

## Synopsys

```
# tap and install
$ brew tap hagatasdelus/tap
$ brew install <formula>

# install directly
$ brew install hagatasdelus/tap/<formula>
```
